# Definition of Done — AutobanCV

**Чек-лист производственной готовности системы отслеживания нарушителей платы за проезд:**

1. **Исходные данные валидированы:** Собран корпус изображений автомобилей, удалены дублирующиеся записи, при обучении компенсирована неравномерность распределения по маркам и моделям авто.

2. **Обучающие ноутбуки завершены:** Все четыре вычислительных скрипта успешно выполнены без критических ошибок:
   - Детектор транспортного средства на основе YOLOv8
   - Локализатор государственного номера (YOLOv8)
   - Модуль текстового распознавания номерных знаков (OCR)
   - Генератор векторных представлений автомобилей (ResNet-108)

3. **Целевые показатели качества достигнуты:**
   - Детектор ТС: mAP50 не менее 0.92
   - Локализатор номера: mAP50 не менее 0.90
   - Распознавание текста: Accuracy не менее 0.95
   - Векторизация авто: Top-5 точность не менее 0.96

4. **Результаты моделей архивированы:** Сохранены веса в формате `.pt` и `.pth`, созданы оптимизированные версии для быстрого инференса (`.onnx`, `.engine`), размещены в папке `/reports/models` с ссылкой в документации.

5. **Валидация входящих данных реализована:** Предварительная обработка отклоняет некорректные файлы (неподдерживаемый формат `image/jpeg` и `image/png`, разрешение < `256×256`, отсутствие объектов), с записью диагностики в логи.

6. **Кодовая база протестирована:** Контейнеризированный сервис собирается без ошибок компиляции, автоматизированные проверки дают покрытие >= 85%.

7. **Системная интеграция подтверждена:** Эндпоинт API принимает фотографию, последовательно запускает детекцию → распознавание → векторизацию → поиск в индексе, возвращает 5 наиболее похожих записей с оценками совпадения. Интерфейс оператора выводит результаты с наложением визуальной информации.

8. **Управление рисками (Risk Management):** Все операции идентификации записываются с timestamp и метаданными. Автоматических действий (блокировка, штраф) не происходит — требуется явное согласие оператора системы.

9. **Инструментирование и наблюдаемость:** Система логирует длительность каждого этапа обработки, уровни confidence для каждого результата, ошибки на любом шаге. Интегрирована с Prometheus для сбора метрик, настроены визуализации в Grafana, определены пороги алертов (время обработки > 2 сек, средний confidence < 0.65).

10. **Восстановление после сбоев (Disaster Recovery):** Реализован переключатель для отключения векторного поиска с откатом к базовому режиму (только распознавание номеров). При обнаружении деградации метрик (< 0.50 confidence для > 20% операций в окне) инициируется откат на предыдущую стабильную версию. Описана пошаговая инструкция для ручного переключения между версиями моделей.

11. **Назначены ответственные лица:** 
    - ML-компонент и качество моделей: Специалист по компьютерному зрению
    - Развертывание и операционная поддержка: Backand специалист
    - Валидация результатов и обучение операторов: Менеджер проекта

12. **Справочная документация завершена:** Подготовлены описания требований продукта, спецификация данных, документация архитектуры инженерного решения, все размещены в папке `/specs`.

## Комплекс проверок

### 1. Функциональные тесты
- **Группа 1 (10 образцов):** Машины с полностью видимыми номерами должны идентифицироваться с первого места рейтинга и оценкой совпадения > 0.95
- **Группа 2 (15 образцов):** Машины с частично скрытыми номерами должны найтись в первых двух местах рейтинга с оценкой >= 0.80
- **Группа 3 (8 образцов):** Сложные условия съемки — система либо находит машину в топ-5, либо возвращает низкий confidence с направлением на ручную проверку

### 2. Проверка на устойчивость к некорректным входным данным
Система отбрасывает поврежденные файлы, изображения неправильного размера, без явных объектов в кадре или с невалидным расположением номера, сохраняя в логах причину отклонения каждого файла.

### 3. Комплексные сквозные тесты (end-to-end)
- Векторная база данных содержит все зарегистрированные машины без дублирования
- Приложение для операторов корректно отображает 5 наиболее вероятных совпадений
